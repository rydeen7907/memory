# メモリ解放ツール (Memory Cleaner)

Python製のシンプルで高機能なメモリ解放・監視ツールです。
システムのメモリ使用状況を可視化し、手動または自動で不要なメモリ領域（ガベージコレクション）を解放します。
CPU使用率の監視機能も追加されました。

## 主な機能

*   **メモリ解放機能**
    *   **強力な解放処理**: Pythonのガベージコレクションに加え、**Windows API (`EmptyWorkingSet`) を使用してシステム全体のプロセスから不要なメモリを解放**します。
    *   **手動解放**: ボタン一つで即座に解放処理を実行し、解放された容量(MB)を表示します。
    *   **自動解放**: 指定した分間隔で定期的にメモリ解放を実行します。
*   **モニタリング機能**
    *   **リアルタイム表示**: メモリと**CPU**の使用率、使用量などをプログレスバーと共に表示します。
    *   **履歴グラフ**: 直近60秒間のメモリ（青）と**CPU（オレンジ）**の使用率推移をグラフ化します。
    *   **上位プロセス表示**: メモリおよび**CPU**を多く消費しているプロセスの上位3件をそれぞれリアルタイムでリストアップします。
*   **タスクトレイ常駐**
    *   ウィンドウを最小化するとタスクトレイ（通知領域）に格納されます。
    *   トレイアイコンは現在のメモリ使用率に応じて動的に変化（バー表示）します。
    *   トレイアイコンの右クリックメニューから「メモリ解放」「タスクマネージャー起動」「終了」が可能です。
    *   トレイ格納時の操作結果はバルーン通知でお知らせします。
*   **ユーティリティ・設定**
    *   **スタートアップ登録**: PC起動時に自動でアプリを実行する設定が可能です。
    *   **最小化起動**: 起動時にウィンドウを表示せず、トレイに格納した状態で開始できます。
    *   **最前面表示**: ウィンドウを常に手前に表示するオプションがあります。
    *   **タスクマネージャー連携**: アプリ内からWindowsタスクマネージャーを直接起動できます。
    *   **警告機能**: 設定した閾値を超えるとウィンドウの背景色が変化し、視覚的に警告します。
    *   **設定保存**: 閾値や自動解放の間隔は `config.json` に自動保存されます。

## 必要要件

*   Python 3.x
*   Windows OS (Windows APIおよびレジストリ操作を使用するため)

### 依存ライブラリ

以下のライブラリが必要です。

```bash
pip install psutil pystray Pillow
```

## 使い方

1.  `memory_cleaner.py` を実行してアプリケーションを起動します。
2.  必要に応じて「警告閾値」や「定期解放の間隔」を設定してください。
3.  ウィンドウを閉じるとアプリが終了しますが、最小化ボタンを押すとタスクトレイに常駐します。
