# Memory Cleaner Tool

Python製のシンプルで軽量なメモリ解放・リソース監視ツールです。
Tkinterを使用したGUIアプリケーションで、システムのメモリ使用率やCPU使用率を監視し、必要に応じてメモリを解放します。

## 機能

*   **リアルタイム監視**:
    *   メモリとCPUの使用率をプログレスバーと数値でリアルタイム表示。
    *   メモリ使用量とCPU使用率が高いプロセス（トップ3）を表示。
*   **グラフ表示**:
    *   メモリとCPUの使用率の推移を別ウィンドウのグラフで可視化。
*   **メモリ解放**:
    *   **手動解放**: ボタンクリックで即座にメモリを解放。
    *   **自動解放**: 設定した間隔（分単位）で自動的にメモリを解放。
    *   **解放ロジック**: Pythonのガベージコレクションに加え、Windows環境ではAPI (`EmptyWorkingSet`) を使用して強力に解放します。
*   **タスクトレイ常駐**:
    *   ウィンドウを最小化するとタスクトレイに格納され、バックグラウンドで動作します。
    *   トレイアイコンのコンテキストメニューから操作可能。
*   **設定**:
    *   **警告閾値**: メモリ使用率が設定値を超えると画面が赤く変化して警告。
    *   **スタートアップ登録**: Windows起動時に自動実行（Windowsのみ）。
    *   **最小化起動**: アプリケーション起動時に最小化状態で開始。
    *   **最前面表示**: ウィンドウを常に最前面に固定。

## 動作環境

*   Python 3.x
*   OS: Windows (推奨), macOS, Linux
    *   ※ メモリ解放の強力な機能（ワーキングセットの削減）やスタートアップ登録はWindows固有の機能です。他OSではPythonのGCのみ動作します。

## 必要ライブラリ

以下のライブラリをインストールしてください。

```bash
pip install psutil pystray pillow
```

※ `tkinter` はPython標準ライブラリですが、Linux環境などでは別途インストールが必要な場合があります（例: `sudo apt install python3-tk`）。

## 実行方法

```bash
python memory_cleaner.py
```

## ファイル構成

アプリケーションは機能ごとにモジュール化されています。

*   `memory_cleaner.py`: アプリケーションのエントリーポイント
*   `ui_builder.py`: GUIウィジェットの構築
*   `graph_monitor.py`: リソースグラフの描画ロジック
*   `process_monitor.py`: プロセス情報の取得・監視
*   `tray_manager.py`: タスクトレイアイコンの管理
*   `settings_window.py`: 設定ウィンドウのUIとロジック
*   `auto_free_scheduler.py`: 定期的なメモリ解放のスケジュール管理
*   `memory_cleaner_logic.py`: メモリ解放のコアロジック
*   `startup_manager.py`: Windowsスタートアップ登録の管理
*   `config_manager.py`: 設定ファイル (`config.json`) の読み書き
*   `os_utils.py`: OS固有のユーティリティ（タスクマネージャー起動など）
*   `icon_data.py`: アプリケーションアイコンデータ（Base64埋め込み）

アイコン画像はコード内に埋め込まれているため、外部の画像ファイルは不要です。
