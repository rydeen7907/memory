=====  
シンプルに作ったつもりのメモリ解放ツール  

[ 説明 ]  
・デフォルトで警告閾値 80％を超えると GUIの背景色が " tomato " に変わる  
    メモリ解放されると " lightblue " で点滅される  
・定期的にメモリ解放をする機能付き (デフォルト間隔＝1分)  
    警告閾値とメモリの定期解放間隔については起動後変更も可能  

[ デスクトップアイコン ]  
Windows内にあるアイコン格納フォルダから借用  

[ ⚠️ 注意 ⚠️ ]  
コード内 47 - 48 行目が反映されない件について  

この2行は、GUI上にメモリ使用率を表示するためのテキストラベルを作成し、  
ウィンドウ内に配置するという重要な役割を担う。  
ですので、アプリケーションを実行すると、この部分は表示されているはず…  

ただ、「表示されていない」と感じられたのは、  
おそらく初期テキストの ... が一瞬で書き換えられてしまうためだと思われる。  

--- プログラムの流れ ---  

1. setup_ui メソッドで、まず text="現在のメモリ使用率: ..." というラベルが作られる  
2. その直後に update_memory_info メソッドが呼び出される  
3. update_memory_info の中で、実際のメモリ使用率が取得され、  
   .config() によってラベルのテキストが  
   メモリ使用率: 50% (16.00 GB / 32.00 GB) のような具体的な情報に更新される  
   この処理が非常に高速に行われるため、初期値の「...」はユーザーの目にはほとんど見えない  
   ※ ただし、使用する機種のスペックによっては表示されることがある
